# gulp-postcss

## 使用方法(关于gulp的使用 注释已经写得不能再清楚了)

#### 1.克隆项目
```console
$ git clone https://github.com/zhalice2011/gulp-postcss.git
```

#### 2.安装依赖
```console
$ node install
```

#### 3.项目运行
```console
$ gulp
```
